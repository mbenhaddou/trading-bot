# RL Trading Bot

Reinforcement Learning Trading Bot with Temporal Abstraction and CI/CD Pipeline.

## Quick Start

```bash
# Copy environment template
cp .env.example .env

# Edit with your API keys
nano .env

# Build and run
docker-compose up
```
